/skillchain v2.1

The recommended way to use AI Design Components. Just describe what you want to build - skillchain handles the rest.

+
Blueprints - Pre-configured chains for dashboards, APIs, RAG pipelines
+
User Preferences - Saves your choices for smart defaults
+
Parallel Loading - Independent skills load concurrently
+
Category Routing - Auto-routes to the right orchestrator
# Install globally
./commands/install-skillchain.sh --global

# Start building
/skillchain dashboard with charts
/skillchain REST API with postgres
/skillchain RAG pipeline with embeddings

# Get help
/skillchain help
